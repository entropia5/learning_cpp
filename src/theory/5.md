Приведение типов

int a = 5;
double b = a;               // неявно: int → double
int c = (int)3.14;          // C-style
int d = static_cast<int>(3.14); // безопаснее
int* p = reinterpret_cast<int*>(0x1234); // адрес → указатель
const int x = 10;
int& y = const_cast<int&>(x); // снимаем const


Неявное — компилятор решает
C-style — «делай что хочешь»
static_cast — безопасный и осознанный
reinterpret_cast — низкоуровневая магия
const_cast — снимаем защиту const
