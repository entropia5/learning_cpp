 2)
Далее у нас есть КЛАССЫ, которые являются конструкцией С++ для моделей.
Классы содержат члены двух типов: данные и функции.

Экземпляр класса - объект.

Если у нас есть три переменные класса, то принято считать, что у нас есть три объекта.
Члены класса могут быть public or private. 
1....Public члены доступны через любую часть кода. С помощью кода вне класса. 
2...Private доступны только внутри класса.
но есть и третий спецификатор доступа. Доступ к защищенным членам можно получать в наследуемых классах, что приводит к наследованию....


3) 

Класс может иметь подклассы, основанные на нём.
Подкласс может наследовать члены от своего суперкласса.


Есть интересная фича связана с наследованием....Это полиморфизм. Суперкласс может определять некоторую функцию, но унаследованная версия этой ф-ции реализуется по - разному среди подклассов, что делает эти родственные классы полиморфными.
