Pointers/ Указатели
Основы
Указатель - особый тип переменных который содержит адрес памяти

int x = 42;      // обычная переменная
int* p = &x;     // указатель хранит адрес x

x хранит 42
p хранит место в памяти, где лежит 42
& → “взять адрес”
* → “разыменовать” (получить значение по адресу)

std::cout << *p;  // 42


int x = 10;
int* ptr = &x; // указатель на x
*ptr = 20;     // меняем x через указатель

&x → адрес переменной
*ptr → значение по адресу


Одним из наиболее важных применений указателей является динамической управление памятью
Что значит? 
Например когда мы объявили переменную - мы выделили для неё статическую память.Компилятор знает какое пространство в памяти потребуется. Однако реальные приложения в основном создают переменные и объекты в памяти во время выполнения. Это и называется динамичесим управлением памятью.

Ссылки / References 
- Это особая категория типом данных. Ссылки являются псевдонимами Существующих переменных.
Поэтому ссылка становится альтернативным именем переменной.
Чтобы объявить ссылку используем нужный тип за которым следует &.




Векторы 

STL векторы.
STL — это часть стандартной библиотеки,
а std::vector — один из контейнеров STL


это контейнеры динамического размера. Это значит что мы можем изменить их размер если нам нужно добавить данные в колекцию или удалить элементы

Память необходимая для векторов, управляется вместо нас. Не не нужно ни выделять ни освобождать память вручную. Объекты которые мы помещаем в свои векторы выделяют память вручную 

Вектор - универсальный класс-контейнер. Значит они могут хранить любой тип элементов, целых чисел, чисел с плавающей запятой, строк или даже собственных классов.
