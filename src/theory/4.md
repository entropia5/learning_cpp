arrays / массивы

Доступ в каждому элементу осуществляется по индексу
Элементы все однородные...то есть относятся к одному и тому же типу данных!

масивы имеют фикс размер. Мы не можем изменять их размер.

Элементы непрерывны в памяти...Значит что весь массив представляет собой последовательный блок адресов памяти

strings / строки
в языке С строки представляют собой массивы символов, то есть последовательность символов, которые заканчиваются на нулевой символ, который кодируется как ноль. 

Класс string / std::string в стандартной библиотеке С++ это модель для строк с большими кол-ством ф-ций.
